Map, Filter, and Reduce Javascript Functions

View CodePen

What is Map?

Map creates a new array by iterating over all elements in an array.

Example: Create a new array after filtering the array based on a custom condition.

What is Filter?

Filter creates a shallow copy of the array, filtered by a custom condition. It does not create a new separate array (deep copy).

Example: Filter an object to only show Users that are over 18. To create a new array (deep copy), use MAP after FILTERING.

What is Reduce?

Reduce iterates through an array and returns a single value.

Example: Get the sum of the users ages.
